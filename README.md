# TAR-CNC-Machine
Quick basic Custom CNC Machine

## Relevant Images
- Image of CAD (yet to come)
- Image of Electronics/PCB (yet to come)

## Main Features
- Automated CNC cutting
- Wireless control from local devices

## Main Hardware/Electronics
- Threaded Ball Screws and Nema 17 Steppers for motion
- Linear rails for stable movement and support
- Aluminum Extrusion for main body
- Raspberry pi for control
